# Ev Fiyat Tahmini (Machine Learning)

Bu proje, metrekare, oda sayısı ve bina yaşına göre
ev fiyatı tahmini yapan bir makine öğrenmesi çalışmasıdır.

## Kullanılan Yöntem
- Linear Regression
- Feature Engineering (yaş için kural tabanlı ceza)

## Özellikler
- Metrekare
- Oda sayısı
- Bina yaşı (kademeli ceza yöntemi)

## Değerlendirme
- MAE (Mean Absolute Error)
- Test MAE ≈ 3.3
